## Interaction Contract

### State & Transitions
- **Drag & Drop:**
  - Active Drag: `opacity-40`, border `border-ares-red`, shadow `shadow-ares-red/20`.
  - Drop Target: Highlight column border with `ares-red/20`.
- **Task Drawer:**
  - Transition: Slide-in from right (`motion.div` x: [100%, 0]).
  - Overlay: `bg-black/60 backdrop-blur-sm`.
- **Feedback:**
  - Saving: Card shows `animate-pulse` or small `RefreshCw` spinner.
  - Presence: Real-time user avatars (Liveblocks) in top-right of board.

### Accessibility Contract
- **Keyboard:**
  - `Tab` navigates cards and column actions.
  - `Space/Enter` opens the Task Drawer for the focused card.
  - `dnd-kit` accessible drag handles with `aria-roledescription="sortable"`.
- **Screen Reader:**
  - Distinct labels for "Edit Task", "Delete Task", and "Drag to reorder".
  - Status changes announced via ARIA live regions if applicable.
- **Contrast:**
  - White text on `ares-red` and `ares-gold` badges must maintain > 4.5:1 ratio.

## Responsive Strategy

| Breakpoint | Layout Strategy |
|------------|-----------------|
| **Mobile (< 640px)** | Single column stack with horizontal scroll (Snap) or Top-level Filter tabs (Todo / Progress / Done). |
| **Tablet (< 1024px)** | 2-column grid or horizontal overflow board. |
| **Desktop (> 1024px)** | 4-column flex grid (equal widths). |
| **Fullscreen** | Expand board to fill viewport, hide sidebar, increase card padding to `md`. |

---

## Data & Logic Contract

### Form Inputs (Task Drawer)
- **Title:** `text`, required, max 100 chars.
- **Description:** `textarea`, markdown supported (Tiptap-lite).
- **Subteam:** `select` (Software, Build, Business, General).
- **Priority:** `select` (Low, Medium, High, Critical).
- **Assignees:** `multi-select` (Team members list).
- **Due Date:** `date-picker`.

### Validation & Errors
- **Sync Error:** "Sync Failed: Your changes were not saved. Retrying..." (Toast).
- **Empty Title:** Prevent "Save Task" if title is whitespace only.
