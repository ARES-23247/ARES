<task type="auto">
  <name>Task 1: Migrate awards.ts and entities.ts to contract inference</name>
  <files>
    functions/api/routes/awards.ts
    functions/api/routes/entities.ts
  </files>
  <action>
For each file (awards.ts, entities.ts):

1. Add import: `import type { AppRouteInput } from "../../../shared/types/contracts";`

2. Update all handler signatures to use inferred types:
   - Change `async ({ body }: { body: any }, c: Context<AppEnv>) =>` to `async (input, c) =>`
   - Change `async ({ query }: { query: any }, c: Context<AppEnv>) =>` to `async (input, c) =>`
   - Change `async ({ params }: { params: any }, c: Context<AppEnv>) =>` to `async (input, c) =>`
   - Update destructuring: `const { field } = input.body;` instead of `const { field } = body;`

3. Remove `as any` cast from router setup:
   - Change `const router: any = s.router(contract as any, handlers)` to `const router = s.router(contract, handlers)`

4. Enable response validation in createHonoEndpoints call:
   ```typescript
   createHonoEndpoints(
     contract,
     router,
     app,
     {
       responseValidation: true,
       responseValidationErrorHandler: (err, _c) => {
         console.error('[Contract] Response validation failed:', err.cause);
         return { error: { message: 'Internal server error' }, status: 500 };
       }
     }
   );
   ```

5. Run TypeScript check: `npx tsc --noEmit` to verify no type errors
  </action>
  <verify>
    <automated>
      # Verify no 'as any' casts in router calls
      grep -n "s\.router.*as any\|: any = s\.router" functions/api/routes/awards.ts functions/api/routes/entities.ts || echo "No 'as any' casts found"

      # Verify responseValidation enabled
      grep "responseValidation: true" functions/api/routes/awards.ts functions/api/routes/entities.ts

      # Verify TypeScript compiles
      npx tsc --noEmit --skipLibCheck 2>&1 | grep -E "(awards|entities)\.ts" | head -5 || echo "No type errors"
    </automated>
  </verify>
  <done>

    - Response validation enabled with error handler
    - TypeScript compilation passes
  </done>
</task>
