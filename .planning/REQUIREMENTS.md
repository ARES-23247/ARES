# Requirements: v4.3 Inquiries, Notifications & Docs Restoration

## Notifications
- [ ] **NOTIF-01**: User notifications must be properly dismissed from the UI and notification bar when marked as read or handled.

## Documentation
- [ ] **DOCS-01**: Missing documentation entries must be recovered or re-created in the database to restore championship-grade data quality.

## Inquiries
- [ ] **INQ-01**: System must automatically generate a Zulip thread for each new inquiry submitted.
- [ ] **INQ-02**: Inquiry details view must include a "Discuss on Zulip" button that links directly to the generated Zulip thread.
- [ ] **INQ-03**: Inquiry view must include a "notes" section, functioning similarly to Kanban card notes, for internal tracking and discussion.

## Out of Scope
- None identified yet.
